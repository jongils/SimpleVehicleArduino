5 Hard Stop 1차 필터 적용
나

6km/h 이하에서 급격히 제동이 되는데 1차 필터 같은걸 넣어서 부드럽게 멈추게 할 수 있나

Claude

1차 지수 감쇠(exponential decay)로 교체합니다. 시상수 HARD_STOP_TAU_S로 속도가 부드럽게 0에 수렴.

const float HARD_STOP_TAU_S = 0.5f; // 감쇠 시상수 [s]

if (gSpeed_ms < HARD_STOP_MS && ...) {
    if (gSpeed_ms < 0.05f) {
        // 최종 스냅
        gSpeed_ms = 0.0f; ...
    } else {
        // 1차 감쇠: v = v * (1 - dt/τ)
        float decay = max(0.0f, 1.0f - dt / HARD_STOP_TAU_S);
        gSpeed_ms *= decay;
        ...
    }
}
HARD_STOP_TAU_S 조정: 낮추면(0.2~0.3) 빠른 정지 / 높이면(0.8~1.0) 더 부드러운 정지
