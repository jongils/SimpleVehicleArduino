6 저속 제동 개선 — Hard Stop
나

제동시 저속 구간에서 차량이 완전히 0km/h가 되지 않는데 개선방법

Claude

근본 원인: explicit Euler 전환 후 저속(vx < ~2 m/s)에서 수치 불안정 발생

안정 조건: dt < 2·J·vx / (Cs·R²) → vx=0.5 m/s 시 dt < 0.19 ms 필요
실제 loop dt ≈ 1~2 ms (약 10배 초과) → omega 진동 → 제동력 불안정

7 Hard Stop 1차 지수 감쇠 필터 적용
나

6km/h 이하에서 급격히 제동이 되는데 1차 필터 같은걸 넣어서 부드럽게 멈추게 할 수 있나

Claude

즉시 스냅 대신 1차 지수 감쇠(exponential decay)로 교체. 아주 느려졌을 때(< 0.05 m/s)에만 최종 스냅.

const float HARD_STOP_TAU_S = 0.5f; // 감쇠 시상수 [s]

if (gSpeed_ms < HARD_STOP_MS && ...) {
    if (gSpeed_ms < 0.05f) {
        gSpeed_ms = 0.0f; // 최종 스냅
        gVy_ms = gYawRate_rads = ... = 0.0f;
    } else {
        float decay = max(0.0f, 1.0f - dt / HARD_STOP_TAU_S);
        gSpeed_ms *= decay;
        wFL.omega = wFR.omega = wRL.omega = wRR.omega = gSpeed_ms / P_WHEEL_R_M;
        gAccel_ms2 = -gSpeed_ms / HARD_STOP_TAU_S;
    }
}
HARD_STOP_TAU_S 조정: 0.2~0.3 → 빠른 정지 / 0.8~1.0 → 더 부드러운 정지

나

vy_dot이랑 ay랑 같은거 아닌지, 그리고 횡속에 대한 변수는 없는지?

Claude

vy_dot ≠ ay

vy_dot = (Fyf + Fyr) / m  -  r · Vx    ← 원심력 항 포함
ay     = (Fyf + Fyr) / m               ← 타이어 힘만

차이: r · Vx (원심가속도 성분). 직진 시 r≈0 이면 거의 같지만 코너링 시 달라집니다.

횡속도 변수: gVy_ms가 line 16에 Vy:로 이미 출력 중입니다.
